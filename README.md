## 🧠 OOP Pillars at a Glance

```
OOP Pillars
├── Encapsulation   → bundling data + methods, hiding data via private
├── Abstraction     → hiding complexity using abstract classes
├── Inheritance     → reusing code via parent-child (IS-A relationship)
└── Polymorphism    → one interface, many forms (overloading + virtual)
```
